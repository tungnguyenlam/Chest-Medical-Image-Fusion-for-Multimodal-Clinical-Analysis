\section{Introduction}
\label{sec:introduction}

Chest radiography is a fundamental imaging modality in clinical practice, particularly in Emergency Departments (ED) and inpatient wards where diagnostic decisions must be made rapidly. It is widely used to detect thoracic conditions such as pneumonia, pneumothorax, pulmonary oedema, pleural effusion, and cardiomegaly \cite{irmici2023cxr_emergency_ai}. However, automated chest X-ray interpretation remains clinically challenging due to overlapping anatomical structures, subtle visual signs, and variations in image quality \cite{irmici2023cxr_emergency_ai}.

In clinical settings, chest radiographs are not interpreted in isolation. Clinicians interpret images alongside clinical context, including indications, active symptoms, vital signs, and historical studies. While deep learning has significantly improved automated chest X-ray classification, many widely used chest X-ray classification baselines are image-centric, using radiographs as the primary input \cite{irvin2019chexpert,johnson2019mimiccxr}. This image-only focus creates a gap between automated systems and clinical workflow, where patient history is essential for accurate diagnosis.

Furthermore, chest X-ray classification is naturally a multi-label, long-tailed problem. Multiple findings often co-occur, and many critical conditions occur far less frequently than common ones, as highlighted by the CXR-LT benchmarks \cite{holste2024cxrlt,holste2025cxrlt2024}. We build on CaMCheX \cite{sloan2025camchex} and test multimodal chest X-ray classification under our available compute budget. Our model fuses radiographs, indication text, vital signs, and prior-study context to better match how clinicians interpret chest X-rays.

\subsection{Background and Motivation}
\label{subsec:intro_background}

Chest X-ray interpretation is a complex diagnostic task where visual findings must be contextualized using a patient's overall clinical state. The significance of a radiographic finding often depends on non-image details: a pulmonary opacity may represent acute pneumonia in a febrile patient with cough, but indicate chronic atelectasis in an asymptomatic patient.

While deep learning has achieved strong diagnostic performance on large datasets like MIMIC-CXR and CheXpert \cite{johnson2019mimiccxr,irvin2019chexpert}, conventional models remain image-centric, limiting their effectiveness in settings where clinical context influences diagnostic urgency. We build a CaMCheX-inspired multimodal model that combines radiographs, clinical text, vital signs, and prior-study context within our compute limits.

\subsection{Task and Dataset Context}
\label{subsec:intro_task_dataset}

The objective is long-tailed, multi-label chest X-ray classification: predicting the presence or absence of multiple thoracic findings from a chest X-ray study. Medical diagnosis is naturally multi-label, as patients frequently present with concurrent conditions (e.g., cardiomegaly, pleural effusion, and pulmonary oedema).

The experimental setup is aligned with the CXR-LT benchmark, which reflects clinical distributions where common conditions appear frequently while rare, critical findings occur infrequently \cite{holste2024cxrlt,holste2025cxrlt2024}. This severe imbalance biases standard models toward frequent labels, necessitating strategies that handle long-tailed class distributions.

Data is sourced from MIMIC-CXR and MIMIC-IV \cite{johnson2019mimiccxr,johnson2024mimiciv,johnson2023mimicived}. We integrate these datasets to obtain study images, clinical indications, structured vital signs, and linked prior studies.

\subsection{Clinical and Technical Challenges}
\label{subsec:intro_challenges}

Developing this multimodal chest X-ray model requires addressing four main challenges:

First, image-only models have limited access to clinical context. Although radiographs contain essential visual evidence, clinical interpretation depends on indications, symptoms, and vital signs to guide diagnostic decisions.

Second, the input modalities are heterogeneous. Reconciling high-dimensional images, free-text indications, and structured numerical vitals requires encoding different formats, feature scales, and missingness patterns into a shared representation.

Third, the classification problem is long-tailed and multi-label. Severe class imbalance and label co-occurrence bias models toward common findings, requiring specialized objectives to maintain sensitivity on rare, critical conditions.

Fourth, prior-study information introduces shortcut and leakage risks. While comparing previous examinations is clinically valuable, historical reports can lead to shortcut learning if the model relies on prior text instead of analyzing the current radiograph.

Finally, multimodal fusion is computationally expensive. High-resolution images generate large token sequences, causing quadratic attention costs in transformer-based architectures \cite{vaswani2017attention} that must be optimized for limited GPU memory.

\subsection{Approach Overview}
\label{subsec:intro_approach}

We adapt CaMCheX for multimodal chest X-ray classification under our compute budget \cite{sloan2025camchex}. It combines current radiographic images, clinical indication text, structured vital signs, and prior-study context into a single classification pipeline.

We encode each modality with a dedicated encoder: a lightweight visual backbone for radiographs, a pre-trained clinical language model for indication text, projection layers for vital signs, and temporal embeddings for prior-study information. We combine these features with a transformer-based fusion module. The model uses compact token layouts and feature routing to balance spatial detail with training efficiency. We use prior-study context as auxiliary temporal evidence, so the model can compare studies without relying entirely on historical information.

\subsection{Thesis Contributions}
\label{subsec:intro_contributions}

The primary contributions are:
\begin{itemize}
    \item A CaMCheX-inspired multimodal chest X-ray classification framework combining radiographs, clinical indication text, structured vital signs, and prior-study context.
    \item Evaluation of practical design choices under limited computational resources, including reduced image resolutions, frozen text encoders, and compact token layouts.
    \item A prior-aware temporal modelling component incorporating historical study information and time-gap embeddings, designed to reduce shortcut and leakage risks.
    \item An ablation study analyzing the contribution of each architectural component, including the label-correlation graph head and channel encoding strategies.
    \item Qualitative interpretability analysis using Grad-CAM visualizations \cite{selvaraju2017gradcam} to inspect visual attention maps against clinical findings.
\end{itemize}

Overall, we present a resource-constrained implementation and analysis of clinically motivated multimodal chest X-ray classification, not a completely new clinical AI framework.
