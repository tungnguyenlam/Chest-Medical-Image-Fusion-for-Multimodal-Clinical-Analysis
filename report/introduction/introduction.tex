\subsection{Research Challenges}
\label{subsec:intro_challenges}

Describe the key challenges that current systems face and that this thesis aims to address:
\begin{itemize}
    \item \textbf{Single-Modality Limitations}: Standard classification models only look at CXRs, missing vital clinical context.
    \item \textbf{Noisy Text Translation}: Existing fusion models translate numeric vitals into natural language, introducing translation noise and relying on imperfect text encoders.
    \item \textbf{Temporal Prior Ignorance}: Diagnostic workflows depend heavily on comparison with previous (prior) exams, yet most models analyze current studies in isolation.
    \item \textbf{Computational Bottlenecks}: Multi-modal token fusion (e.g., in dense self-attention) scales quadratically, making it expensive for real-time clinical deployment.
    \item \textbf{Severe Class Imbalance}: Medical findings exhibit highly skewed distributions, where a few pathologies dominate and many critical conditions are rare.
\end{itemize}

\subsection{Proposed Solution: CaMCheX}
\label{subsec:intro_proposed}

Introduce the Clinically-aligned Multi-modal Chest X-ray Classification framework (\textbf{CaMCheX}). Briefly explain its evolution:
\begin{itemize}
    \item \textbf{CaMCheX V2}: Spatially expanded fusion that aligns text and vitals features across the spatial grid of the X-ray images.
    \item \textbf{CaMCheX V3}: Slimmed single-token fusion which collapses non-image modalities to single tokens to improve computational efficiency.
    \item \textbf{CaMCheX V4}: Asymmetric prior cross-attention that structures temporal modeling as a query-decoder task, preventing prior-label copying and halving attention complexity.
\end{itemize}
Explain how the model integrates current inputs with historical records (vitals, prior labels, prior reports) while preventing label leakage.

\subsection{Thesis Contributions}
\label{subsec:intro_contributions}

Summarize the main contributions of this work:
\begin{enumerate}
    \item Development of a unified multimodal architecture combining CXRs, clinical text, and numeric vitals.
    \item Design of an asymmetric cross-attention mechanism (V4) to perform compute-efficient and leakage-safe temporal comparison with prior studies.
    \item Systematic analysis of token layouts (V2 vs. V3) evaluating the trade-offs of spatial expansion in multi-modal fusion.
    \item Integration of Asymmetric Loss (ASL) with frequency-based weighting to counter extreme label imbalance.
    \item Comprehensive evaluation on a subset of the MIMIC-CXR and MIMIC-IV datasets.
\end{enumerate}
